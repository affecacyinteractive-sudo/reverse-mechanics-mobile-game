## INT-TRANSFORM — Final (LLM-only, chunk-first, no rigid JSON)

### Purpose

Transform a messy user ask into a **runnable intent pack** that can drive the next 1–3 actions. This is for **human consumption and reuse**, not computer execution, so the contract is **chunk-first** (fixed chunk titles), not JSON.

* **Constraints** — short “must / must not” statements that bound scope and prevent regressions.
* **Done Signal** — observable proof statement: what would convince a reader it worked.

Optional chunks (only if needed):

* **Non-Goals** — explicit exclusions that keep the next action winnable.
* **Assumptions** — only if necessary; each line begins with `Assumed:` (no questions).

Prohibitions:

Aim should be specific enough to judge success. Surface should be concrete and stable (“login flow”, “filter panel state”, “cache invalidation rule”), not a vague subsystem. Constraints should be true boundaries (compatibility, non-regression, scope) rather than aspirations. Done Signal should be phrased like a checkable receipt, not a vibe.

## CODE-COMMIT — Final (latest invariants, hybrid, high reliability)

This is the **one** non-standard action that **must** use a rigid, key-level contract, because the computer needs to apply it to the canon codebase.

### Purpose

Turn patch outputs (often local snippets) into **canonical file updates** by making the LLM produce a strict **CommitPlan**, then applying it mechanically to `CodebaseStore` and emitting an authoritative receipt from what was actually written.

---

## Input Grammar (UI layer)

Commit **[Patch Payload : Targets]** into **[Destination Identity : Targets]**.
If present, align with **[Commit Aim : Intent]**, reconstruct from **[Pre-Commit Slice : Packets]**, and respect **[Recent Context : Chronicle]**.

Required blanks: **Patch Payload**, **Destination Identity**.
Optional blanks: **Commit Aim**, **Pre-Commit Slice**, **Recent Context**.

---

## Output Contract (rigid, hard)

### LLM output must include one machine-consumable chunk

The LLM must emit **one TEXT chunk** with:

* `title` exactly: `CommitPlan`
* `body` exactly: **raw JSON** (no fences, no commentary, no markdown)

Schema:

```json
{
  "run_thesis": "short plain text",
  "file_updates": [
    { "file_path": "string", "new_body": "string" }
  ],
  "assumptions": ["Assumed: ..."]
}
```

Rules:

* `file_updates` must be non-empty.
* Each `new_body` is the **full file body** (Patch Protocol V1 = REPLACE_FILE).
* `file_path` must be a real project path (no placeholders).
* `assumptions` is optional; if present, each line starts with `Assumed:`.

### Deterministic outputs (authoritative)

After applying the plan, the engine returns **chunk packs**:

* a **Commit receipt** TEXT chunk with touched paths and before→after version lineage (canonical truth),
* an optional **Post-commit slice** (CODE chunks titled by file path) for immediate proof/targeting.

---

## LLM runtime prompt text (minimal, contract-only)

You are CODE-COMMIT (Commit Planner). Produce a mechanically appliable commit plan.

Inputs:

* Intent[] is the aim/constraints (why this commit exists).
* Targets[] contains destination identity and patch payload (snippets/framing).
* Packets[] contains the pre-commit code slice. Use it to reconstruct full file bodies.
* Chronicle[] is optional context.

Output Contract:
Emit a TEXT chunk with title exactly "CommitPlan".
Its body must be raw JSON (no fences, no extra text) matching:
{
"run_thesis": "...",
"file_updates": [{"file_path":"...","new_body":"..."}],
"assumptions": ["Assumed: ..."]
}

Rules:

* file_updates must be non-empty.
* new_body must be the FULL file body for that file.
* Use file paths and identifiers that are consistent with Targets/Packets.
* Keep changes minimal and local to the intended surfaces.
* Do not ask questions; if underspecified, choose the smallest reasonable interpretation and record it under assumptions.

import type { RunContainer, RunOutputs } from "./run_container";

  system_prompt: string;

  role_fills?: RunContainer["role_fills"];
  prose_variant?: RunContainer["prose_variant"];

export const CODE_COMMIT_INPUT_GRAMMAR = {

// -------- strict extraction/parsing helpers --------

function findCommitPlanChunk(chunks: Chunk[]): Chunk | null {
  return chunks.find((c) => c.kind === "TEXT" && c.title.trim() === "CommitPlan") ?? null;
}

/**
 * High-reliability JSON extraction:
 * - trims
 * - strips ```json fences if present
 * - extracts the first top-level {...} block if extra text leaked
 */
function extractJsonObject(text: string): string | null {
  let s = text.trim();

  // strip code fences if the model violated the rule
  if (s.startsWith("```")) {
    s = s.replace(/^```[a-zA-Z]*\s*/, "").replace(/```$/, "").trim();
  }

  const first = s.indexOf("{");
  const last = s.lastIndexOf("}");
  if (first < 0 || last < 0 || last <= first) return null;

  return s.slice(first, last + 1);
}

function parseCommitPlanFromChunks(chunks: Chunk[]): { ok: true; plan: CommitPlan } | { ok: false; error: string } {
  const c = findCommitPlanChunk(chunks);
  if (!c) return { ok: false, error: 'Missing required TEXT chunk titled "CommitPlan".' };

  const jsonText = extractJsonObject(c.body);
  if (!jsonText) return { ok: false, error: "CommitPlan body did not contain a parseable JSON object." };

  let obj: any;
  try {
    obj = JSON.parse(jsonText);
  } catch {
    return { ok: false, error: "CommitPlan JSON could not be parsed." };
  }

  if (!obj || typeof obj !== "object") return { ok: false, error: "CommitPlan must be a JSON object." };
  if (typeof obj.run_thesis !== "string" || !obj.run_thesis.trim())
    return { ok: false, error: "CommitPlan.run_thesis must be a non-empty string." };

  if (!Array.isArray(obj.file_updates) || obj.file_updates.length === 0)
    return { ok: false, error: "CommitPlan.file_updates must be a non-empty array." };

  for (const u of obj.file_updates) {
    if (!u || typeof u !== "object") return { ok: false, error: "Each file_update must be an object." };
    if (typeof u.file_path !== "string" || !u.file_path.trim())
      return { ok: false, error: "Each file_update.file_path must be a non-empty string." };
    if (typeof u.new_body !== "string" || !u.new_body.trim())
      return { ok: false, error: "Each file_update.new_body must be a non-empty string (full file body)." };
  }

  if (obj.assumptions !== undefined) {
    if (!Array.isArray(obj.assumptions)) return { ok: false, error: "CommitPlan.assumptions must be an array if present." };
    for (const a of obj.assumptions) {
      if (typeof a !== "string") return { ok: false, error: "CommitPlan.assumptions entries must be strings." };
    }
  }

  return { ok: true, plan: obj as CommitPlan };
}

// -------- deterministic apply + receipts --------

function applyPlanToCodebase(args: { codebase: CodebaseStore; run_id: RunId; plan: CommitPlan }): { touched: string[]; receipt: Chunk[] } {
  const { codebase, run_id, plan } = args;

  const touched: string[] = [];
  const lineage: string[] = [];

  for (const u of plan.file_updates) {
    const file_path = u.file_path.replaceAll("\\", "/").trim();

    const before = codebase.getLatestVersion(file_path);
    const before_id = before?.version_id ?? "∅";

    const after = codebase.upsertFile({ file_path, run_id, body: u.new_body });
    const after_id = after.version_id;

    touched.push(file_path);
    lineage.push(`${file_path}: ${before_id} → ${after_id}`);
  }

  const receiptBody =
    `Run: ${run_id}\n` +
    `Thesis: ${plan.run_thesis}\n` +
    `Touched: ${touched.join(", ")}\n` +
    `Lineage:\n${lineage.join("\n")}`;

  const receipt: Chunk[] = [
    { id: `cr_${Date.now()}_${Math.random()}`, title: "Commit receipt", body: receiptBody, kind: "TEXT" },
  ];

  if (plan.assumptions?.length) {
    receipt.push({
      id: `cr_${Date.now()}_${Math.random()}`,
      title: "Commit assumptions",
      body: plan.assumptions.join("\n"),
      kind: "TEXT",
    });
  }

  return { touched: [...new Set(touched)], receipt };
}

function buildPostCommitSlice(args: { codebase: CodebaseStore; run_id: RunId; file_paths: string[] }): Chunk[] {
  const { codebase, run_id, file_paths } = args;

  const chunks: Chunk[] = [
    {
      id: `ps_${Date.now()}_${Math.random()}`,
      title: "Post-commit slice",
      body: `Run: ${run_id}\nFiles: ${file_paths.join(", ")}`,
      kind: "TEXT",
    },
  ];

  const MAX_LINES = 120;
  for (const fp of file_paths) {
    const v = codebase.getVersionAtRun({ file_path: fp, run_id, inclusive: true });
    if (!v || v.is_deleted) continue;

    const excerpt = v.body.split("\n").slice(0, MAX_LINES).join("\n").trimEnd();
    chunks.push({ id: `ps_${Date.now()}_${Math.random()}`, title: fp, body: excerpt, kind: "CODE" });
  }

  return chunks;
}

// -------- callable action --------

export async function run_CODE_COMMIT(args: { rc: RunContainer; ctx: CodeCommitCtx }): Promise<RunContainer> {
  const { rc, ctx } = args;

  // Phase A: LLM CommitPlan
  const planned = await ctx.invokeLLM({
    action_id: "CODE-COMMIT",
    system_prompt: CODE_COMMIT_SYSTEM_PROMPT,
    inputs: rc.inputs,
    role_fills: rc.role_fills,
    prose_variant: rc.prose_variant,
  });

  const parsed = parseCommitPlanFromChunks(planned.chunks);
  if (!parsed.ok) {
    const outputs: RunOutputs = {
      chunks: [
        ...planned.chunks,
        { id: `cf_${Date.now()}_${Math.random()}`, title: "Commit failed", body: parsed.error, kind: "TEXT" },
      ],
      patch_segments: [],
      seals: [],
    };
    return { ...rc, outputs };
  }

  // Phase B: deterministic canon write + authoritative receipt
  const { touched, receipt } = applyPlanToCodebase({ codebase: ctx.codebase, run_id: rc.run_id, plan: parsed.plan });
  const postSlice = buildPostCommitSlice({ codebase: ctx.codebase, run_id: rc.run_id, file_paths: touched });

  const outputs: RunOutputs = {
    chunks: [...planned.chunks, ...receipt, ...postSlice],
    patch_segments: [],
    seals: [],
  };

  return { ...rc, outputs };
}
````

## JOURNAL-APPEND — Final (deterministic, chunk-first, minimal rigid result)

### Purpose

Persist the run’s keystone into canon and append a chronicle entry that references it. This makes the journey memory **real, replayable, and queryable**, without relying on the LLM for the fact of insertion.

This action is **deterministic**. It does not generate meaning; it records meaning that already exists in the provided keystone pack.

---

## Input Grammar (UI layer)

Record **[Keystone Pack : Targets]** into the chronicle as **[School Label : Intent]**.
Optionally, bias anchor selection using **[Anchor Clues : Targets]**.

Required blanks: **Keystone Pack**, **School Label**.
Optional blanks: **Anchor Clues**.

---

## Output Contract (chunk-first, hard)

Output a **Journal Append Receipt Pack** as TEXT chunks with the following exact titles:

Required:

* **JournalAppendResult** — machine-friendly key-value lines (strict format below).
* **Keystone Stored** — human-readable confirmation.
* **Chronicle Entry Added** — human-readable confirmation (journal line + anchors).

Optional:

* **Notes** — only if something was partially missing or clamped (e.g., “anchors trimmed to 3”).

If the keystone pack is missing/empty, output a single TEXT chunk titled **Journal append failed** and do not mutate canon.

### JournalAppendResult strict body format

The body must be exactly these lines (in any order is fine, but each must appear once):

* `run_id: ...`
* `school: ...` (FI/FU/FPR/FA/FS/FP)
* `keystone_id: ...`
* `entry_id: ...`

This is the only “rigid” part: it lets the harness chain follow-up actions without guesswork.

---

## Execution Semantics (deterministic)

* Determine `school` from the School Label intent (FI/FU/FPR/FA/FS/FP). If unclear, default to `FI`.
* Store keystone chunks into `KeystoneStore` → `keystone_id`.
* Derive `journal_line` from the **Keystone Body** chunk if present; otherwise fall back to first TEXT chunk body. Use the first sentence, clamp length.
* Derive `anchors`:

  * If the keystone pack contains an **Anchors** chunk, use up to 3 anchors from it.
  * Else, if Anchor Clues are provided, keep up to 3 that appear in keystone text.
  * Else extract up to 3 identifier-like tokens from keystone text.
* Append chronicle entry into `ChronicleStore` → `entry_id`.

No LLM calls, no invention.

---

# Deterministic action script (callable like an action, includes Input Grammar)

```ts
// action_journal_append.ts — JOURNAL-APPEND (deterministic) V1
// Depends on: types.ts, run_container.ts, chronicle_store.ts

import type { Chunk, SchoolId } from "./types";
import type { RunContainer, RunOutputs } from "./run_container";
import { KeystoneStore, ChronicleStore } from "./chronicle_store";

export type JournalAppendCtx = {
  keystones: KeystoneStore;
  chronicle: ChronicleStore;
};

export const JOURNAL_APPEND_INPUT_GRAMMAR = {
  simple: `Record [Keystone Pack : Targets] into the chronicle as [School Label : Intent].`,
  pro: `Record [Keystone Pack : Targets] into the chronicle as [School Label : Intent], biasing anchors with [Anchor Clues : Targets].`,
};

function schoolFromIntent(intent: Chunk[]): SchoolId {
  const text = intent.map((c) => `${c.title}\n${c.body}`).join("\n").toUpperCase();
  if (text.includes("FPR")) return "FPR";
  if (text.includes("FU")) return "FU";
  if (text.includes("FA")) return "FA";
  if (text.includes("FS")) return "FS";
  if (text.includes("FP")) return "FP";
  if (text.includes("FI")) return "FI";
  return "FI";
}

function clamp(s: string, max: number): string {
  return s.length <= max ? s : s.slice(0, max - 1).trimEnd() + "…";
}

function firstSentence(text: string): string {
  const t = text.trim();
  if (!t) return "";
  const s = t.split(/(?<=[.!?])\s+/)[0] ?? t;
  return s.trim();
}

function getKeystoneBodyText(keystoneChunks: Chunk[]): string {
  const bodyChunk = keystoneChunks.find((c) => c.kind === "TEXT" && c.title.trim() === "Keystone Body");
  if (bodyChunk?.body?.trim()) return bodyChunk.body;
  const anyText = keystoneChunks.find((c) => c.kind === "TEXT" && c.body.trim());
  return anyText?.body ?? "";
}

function parseAnchorsChunk(keystoneChunks: Chunk[]): string[] {
  const c = keystoneChunks.find((x) => x.kind === "TEXT" && x.title.trim() === "Anchors");
  if (!c) return [];
  // one per line preferred; also accept comma-separated
  return c.body
    .split(/[\n,]/)
    .map((s) => s.trim())
    .filter(Boolean);
}

function parseAnchorClues(targets: Chunk[]): string[] {
  // Optional convention: TEXT chunk titled exactly "Anchor Clues"
  const c = targets.find((x) => x.kind === "TEXT" && x.title.trim() === "Anchor Clues");
  if (!c) return [];
  return c.body
    .split(/[\n,]/)
    .map((s) => s.trim())
    .filter(Boolean);
}

function deriveAnchors(args: { keystoneChunks: Chunk[]; anchorClues: string[] }): string[] {
  const { keystoneChunks, anchorClues } = args;

  const keystoneText = keystoneChunks
    .filter((c) => c.kind === "TEXT")
    .map((c) => `${c.title}\n${c.body}`)
    .join("\n");

  // 1) Keystone-provided Anchors chunk wins
  const fromAnchorsChunk = parseAnchorsChunk(keystoneChunks)
    .map((a) => a.trim())
    .filter(Boolean)
    .slice(0, 3);

  if (fromAnchorsChunk.length) return dedupe(fromAnchorsChunk).slice(0, 3);

  // 2) Anchor clues next, but only if they appear in keystone text
  const fromClues: string[] = [];
  for (const clue of anchorClues) {
    if (keystoneText.includes(clue) && !fromClues.includes(clue)) fromClues.push(clue);
    if (fromClues.length >= 3) break;
  }
  if (fromClues.length) return fromClues;

  // 3) Fallback: identifier-like token harvest
  const candidates = new Set<string>();
  const re = /(#\w+|[A-Z][a-zA-Z0-9]+|[a-z]+[A-Z][a-zA-Z0-9]*|[a-z0-9_]{3,})\b/g;

  let m: RegExpExecArray | null;
  while ((m = re.exec(keystoneText))) {
    const tok = m[0];
    if (tok.length < 3) continue;
    candidates.add(tok);
    if (candidates.size >= 24) break;
  }

  return [...candidates].slice(0, 3);
}

function dedupe(xs: string[]): string[] {
  return [...new Set(xs.map((x) => x.trim()).filter(Boolean))];
}

export function run_JOURNAL_APPEND(args: { rc: RunContainer; ctx: JournalAppendCtx }): RunContainer {
  const { rc, ctx } = args;

  // Keystone pack must be present in Targets
  const keystonePack = rc.inputs.Targets.filter((c) => c.kind === "TEXT" || c.kind === "CODE");
  if (keystonePack.length === 0) {
    const outputs: RunOutputs = {
      chunks: [
        {
          id: `ja_${Date.now()}_${Math.random()}`,
          title: "Journal append failed",
          body: "No keystone chunks were provided in Targets, so nothing was recorded.",
          kind: "TEXT",
        },
      ],
      patch_segments: [],
      seals: [],
    };
    return { ...rc, outputs };
  }

  const school = schoolFromIntent(rc.inputs.Intent);

  // Store keystone
  const keystone = ctx.keystones.putKeystone({
    run_id: rc.run_id,
    school,
    chunks: keystonePack,
  });

  // Derive journal_line + anchors deterministically
  const kb = getKeystoneBodyText(keystonePack);
  const journal_line = clamp(firstSentence(kb) || "Recorded keystone.", 160);

  const anchorClues = parseAnchorClues(rc.inputs.Targets);
  const anchors = deriveAnchors({ keystoneChunks: keystonePack, anchorClues });

  // Append chronicle entry
  const entry = ctx.chronicle.appendEntry({
    run_id: rc.run_id,
    school,
    keystone_id: keystone.keystone_id,
    journal_line,
    anchors,
  });

  const resultKv =
    `run_id: ${rc.run_id}\n` +
    `school: ${school}\n` +
    `keystone_id: ${keystone.keystone_id}\n` +
    `entry_id: ${entry.entry_id}`;

  const outputs: RunOutputs = {
    chunks: [
      {
        id: `ja_${Date.now()}_${Math.random()}`,
        title: "JournalAppendResult",
        body: resultKv,
        kind: "TEXT",
      },
      {
        id: `ja_${Date.now()}_${Math.random()}`,
        title: "Keystone Stored",
        body: `Stored keystone ${keystone.keystone_id} for run ${rc.run_id} as ${school}.`,
        kind: "TEXT",
      },
      {
        id: `ja_${Date.now()}_${Math.random()}`,
        title: "Chronicle Entry Added",
        body:
          `Added chronicle entry ${entry.entry_id} referencing keystone ${keystone.keystone_id}.\n` +
          `Journal line: ${entry.journal_line}\n` +
          `Anchors: ${entry.anchors.join(", ") || "∅"}`,
        kind: "TEXT",
      },
    ],
    patch_segments: [],
    seals: [],
  };

  return { ...rc, outputs };
}
```

* Emit SliceResult + Slice Header + CODE chunks for included files.

# Deterministic action script (callable like an action, includes Input Grammar)

```ts
// action_code_slice.ts — CODE-SLICE (deterministic) V1
// Depends on: types.ts, run_container.ts, codebase_store.ts

import type { Chunk } from "./types";
import type { RunContainer, RunOutputs } from "./run_container";
import { CodebaseStore } from "./codebase_store";

export type CodeSliceCtx = {
  codebase: CodebaseStore;
};

export const CODE_SLICE_INPUT_GRAMMAR = {
  simple: `Extract a grounding slice for [Slice Mission : Intent] from [Extraction Target : Targets].`,
  pro: `Extract a grounding slice for [Slice Mission : Intent] from [Extraction Target : Targets], prioritizing [Primary File Hints : Targets] and applying [Slice Preference : Intent].`,
};

function extractFilePathsFromChunks(chunks: Chunk[]): string[] {
  const out = new Set<string>();
  const re =
    /([A-Za-z0-9_\-./\\]+?\.(ts|tsx|js|jsx|py|java|cs|go|rs|sql|json|yaml|yml|toml|md|txt))(::[A-Za-z0-9_\-./#]+)?/g;

  for (const c of chunks) {
    if (c.kind !== "TEXT") continue;
    let m: RegExpExecArray | null;
    while ((m = re.exec(c.body))) out.add(m[1].replaceAll("\\", "/"));
  }
  return [...out];
}

function dedupe(xs: string[]): string[] {
  return [...new Set(xs.map((x) => x.trim()).filter(Boolean))];
}

export function run_CODE_SLICE(args: { rc: RunContainer; ctx: CodeSliceCtx }): RunContainer {
  const { rc, ctx } = args;

  const candidates = dedupe(extractFilePathsFromChunks(rc.inputs.Targets));
  if (candidates.length === 0) {
    const outputs: RunOutputs = {
      chunks: [
        {
          id: `sl_${Date.now()}_${Math.random()}`,
          title: "Slice failed",
          body: "No resolvable file paths were found in Targets, so no code could be extracted.",
          kind: "TEXT",
        },
      ],
      patch_segments: [],
      seals: [],
    };
    return { ...rc, outputs };
  }

  const included: string[] = [];
  const missing: string[] = [];
  const deleted: string[] = [];

  const codeChunks: Chunk[] = [];
  const MAX_LINES_PER_FILE = 120;

  for (const fp of candidates) {
    const v = ctx.codebase.getVersionAtRun({ file_path: fp, run_id: rc.run_id, inclusive: true });

    if (!v) {
      missing.push(fp);
      continue;
    }
    if (v.is_deleted) {
      deleted.push(fp);
      continue;
    }

    included.push(fp);

    const excerpt = v.body.split("\n").slice(0, MAX_LINES_PER_FILE).join("\n").trimEnd();
    codeChunks.push({
      id: `sl_${Date.now()}_${Math.random()}`,
      title: fp,
      body: excerpt,
      kind: "CODE",
    });
  }

  if (included.length === 0) {
    const outputs: RunOutputs = {
      chunks: [
        {
          id: `sl_${Date.now()}_${Math.random()}`,
          title: "Slice failed",
          body:
            `No files could be extracted at run ${rc.run_id}.\n` +
            `Missing: ${missing.join(", ") || "∅"}\n` +
            `Deleted: ${deleted.join(", ") || "∅"}`,
          kind: "TEXT",
        },
      ],
      patch_segments: [],
      seals: [],
    };
    return { ...rc, outputs };
  }

  const resultKv =
    `run_id: ${rc.run_id}\n` +
    `included: ${included.join(", ")}\n` +
    `missing: ${missing.join(", ") || "∅"}\n` +
    `deleted: ${deleted.join(", ") || "∅"}`;

  const mission = rc.inputs.Intent[0]?.body?.trim() || "Grounding slice";
  const headerText =
    `Mission: ${mission}\n` +
    `Included: ${included.join(", ")}\n` +
    (missing.length ? `Missing: ${missing.join(", ")}\n` : "") +
    (deleted.length ? `Deleted: ${deleted.join(", ")}\n` : "");

  const outputs: RunOutputs = {
    chunks: [
      { id: `sl_${Date.now()}_${Math.random()}`, title: "SliceResult", body: resultKv, kind: "TEXT" },
      { id: `sl_${Date.now()}_${Math.random()}`, title: "Slice Header", body: headerText.trimEnd(), kind: "TEXT" },
      ...codeChunks,
    ],
    patch_segments: [],
    seals: [],
  };

  return { ...rc, outputs };
}
```

// Depends on: types.ts, run_container.ts, chronicle_store.ts

import type { Chunk } from "./types";

import { ChronicleStore } from "./chronicle_store";

export const JOURNEY_VIEW_INPUT_GRAMMAR = {
  simple: `Render the recent journey for [View Purpose : Intent] using [Chronicle Window : Chronicle].`,
  pro: `Render the recent journey for [View Purpose : Intent] using [Chronicle Window : Chronicle], applying [Filter Hint : Intent] and highlighting [Pinned Keystone(s) : Targets].`,

function deriveLimitFromIntent(intent: Chunk[], fallback: number): number {

        },

      seals: [],

  const resultKv =

  const strip = entries
    .map((e) => {
      const a = e.anchors.length ? ` — ${e.anchors.join(", ")}` : "";
      return `[${e.school}] ${e.journal_line}${a}`;
    })
    .join("\n");

  const chunks: Chunk[] = [
    { id: `jv_${Date.now()}_${Math.random()}`, title: "JourneyViewResult", body: resultKv, kind: "TEXT" },
    { id: `jv_${Date.now()}_${Math.random()}`, title: "View Header", body: headerText, kind: "TEXT" },
    { id: `jv_${Date.now()}_${Math.random()}`, title: "Entry Strip", body: strip, kind: "TEXT" },
  ];

  const outputs: RunOutputs = { chunks, patch_segments: [], seals: [] };
  return { ...rc, outputs };
}

# Deterministic action script (callable like an action, includes Input Grammar)

```ts
// action_code_placement_confirm.ts — CODE-PLACEMENT-CONFIRM (deterministic) V1
// Depends on: types.ts, run_container.ts

import type { Chunk } from "./types";
import type { RunContainer, RunOutputs } from "./run_container";

export const CODE_PLACEMENT_CONFIRM_INPUT_GRAMMAR = {
  simple: `Confirm where [Commit Receipt : Targets] landed using [Post-Commit Slice : Packets].`,
  pro: `Confirm where [Commit Receipt : Targets] landed using [Post-Commit Slice : Packets], emphasizing [Confirmation Focus : Intent].`,
};

function parseTouchedFromReceipt(targets: Chunk[]): string[] {
  const receipt = targets.find(
    (c) => c.kind === "TEXT" && c.title.toLowerCase().includes("commit receipt")
  );
  if (!receipt) return [];

  const m = /Touched:\s*(.*)/.exec(receipt.body);
  if (!m) return [];

  return m[1]
    .split(",")
    .map((s) => s.trim())
    .filter(Boolean);
}

function looksLikeFilePath(s: string): boolean {
  const t = s.trim();
  return /\.[A-Za-z0-9]+$/.test(t) && (t.includes("/") || t.includes("\\") || t.startsWith("src") || t.startsWith("app"));
}

function inferTouchedFromPackets(packets: Chunk[]): string[] {
  const out: string[] = [];
  for (const c of packets) {
    if (c.kind !== "CODE") continue;
    if (looksLikeFilePath(c.title)) out.push(c.title.trim().replaceAll("\\", "/"));
  }
  return [...new Set(out)];
}

function pickProofChunks(args: { packets: Chunk[]; touched: string[] }): Chunk[] {
  const { packets, touched } = args;

  const MAX_LINES = 60;
  const proof: Chunk[] = [];

  for (const fp of touched) {
    const src = packets.find((c) => c.kind === "CODE" && c.title.trim().replaceAll("\\", "/") === fp);
    if (!src) continue;

    const excerpt = src.body.split("\n").slice(0, MAX_LINES).join("\n").trimEnd();

    proof.push({
      id: `pc_${Date.now()}_${Math.random()}`,
      title: `Proof: ${fp}`,
      body: excerpt,
      kind: "CODE",
    });
  }

  return proof;
}

export function run_CODE_PLACEMENT_CONFIRM(args: { rc: RunContainer }): RunContainer {
  const { rc } = args;

  const touched = parseTouchedFromReceipt(rc.inputs.Targets);
  const finalTouched = touched.length ? touched : inferTouchedFromPackets(rc.inputs.Packets);

  if (!finalTouched.length) {
    const outputs: RunOutputs = {
      chunks: [
        {
          id: `pc_${Date.now()}_${Math.random()}`,
          title: "Placement unknown",
          body: "Could not confirm touched file paths from commit receipt or post-commit packet slice.",
          kind: "TEXT",
        },
      ],
      patch_segments: [],
      seals: [],
    };
    return { ...rc, outputs };
  }

  const resultKv =
    `run_id: ${rc.run_id}\n` +
    `touched: ${finalTouched.join(", ")}`;

  const summary = `Confirmed placement for run ${rc.run_id}.\nTouched: ${finalTouched.join(", ")}`;

  const chunks: Chunk[] = [
    { id: `pc_${Date.now()}_${Math.random()}`, title: "PlacementConfirmResult", body: resultKv, kind: "TEXT" },
    { id: `pc_${Date.now()}_${Math.random()}`, title: "Placement Summary", body: summary, kind: "TEXT" },
  ];

  const proof = pickProofChunks({ packets: rc.inputs.Packets, touched: finalTouched });
  if (proof.length) {
    chunks.push({
      id: `pc_${Date.now()}_${Math.random()}`,
      title: "Proof Excerpts",
      body: "Included proof excerpts from the post-commit slice for the touched files.",
      kind: "TEXT",
    });
    chunks.push(...proof);
  }

  const outputs: RunOutputs = { chunks, patch_segments: [], seals: [] };
  return { ...rc, outputs };
}
```

## JOURNAL-PLACEMENT-CONFIRM — Final (deterministic, chunk-first, minimal rigid result)

### Purpose

Confirm where a keystone appears in the canon chronicle and show its local neighborhood. This is the chronicle-side “where did it land” proof instrument.

---

## Input Grammar (UI layer)

Confirm where **[Keystone Identity : Targets]** appears in the chronicle, and show neighborhood based on **[Neighborhood Preference : Intent]**.

Required blanks: **Keystone Identity**, **Neighborhood Preference**.
Optional blanks: **Anchor Clues**.

---

## Output Contract (chunk-first, hard)

Output a **Chronicle Placement Pack** as TEXT chunks with the following exact titles:

Required:

* **ChroniclePlacementResult** (TEXT) — machine-friendly key-value lines:

  * `run_id: ...`
  * `keystone_id: ...`
  * `found: true|false`
  * `entry_id: ...` (or `∅`)
* **Matched Entry** (TEXT) — the matched chronicle line (school + journal_line + anchors) if found, otherwise a short “not found” message.

Optional (only if found):

* **Neighborhood** (TEXT) — surrounding entries (previous/next) in the local window.
* **Match Basis** (TEXT) — one short statement: “matched by keystone_id reference”.

Failure case:

* If no `keystone_id` can be parsed from Targets, output a single TEXT chunk titled **Placement not found**.

No invention: lookup is by exact keystone_id match only.

---

## Execution Semantics (deterministic)

* Parse `keystone_id` from Targets deterministically.
* Query `ChronicleStore` for the most recent entry with that `keystone_id`.
* Render result KV + matched entry + neighborhood window (span default 1; can be adjusted deterministically from intent).

---

# Deterministic action script (callable like an action, includes Input Grammar)

```ts id="4coozh"
// action_journal_placement_confirm.ts — JOURNAL-PLACEMENT-CONFIRM (deterministic) V1
// Depends on: types.ts, run_container.ts, chronicle_store.ts

import type { Chunk } from "./types";
import type { RunContainer, RunOutputs } from "./run_container";
import { ChronicleStore } from "./chronicle_store";

export type JournalPlacementCtx = {
  chronicle: ChronicleStore;
};

export const JOURNAL_PLACEMENT_CONFIRM_INPUT_GRAMMAR = {
  simple: `Confirm where [Keystone Identity : Targets] appears in the chronicle, using [Neighborhood Preference : Intent].`,
  pro: `Confirm where [Keystone Identity : Targets] appears in the chronicle, using [Neighborhood Preference : Intent], sanity-checking with [Anchor Clues : Targets].`,
};

function parseKeystoneIdFromTargets(targets: Chunk[]): string | null {
  const labeled = targets.find((c) => c.kind === "TEXT" && c.body.toLowerCase().includes("keystone_id"));
  if (labeled) {
    const m = /keystone_id\s*:\s*([A-Za-z0-9_\-]+)/i.exec(labeled.body);
    if (m) return m[1].trim();
  }

  const bare = targets.find((c) => c.kind === "TEXT" && /^k_[A-Za-z0-9]+/i.test(c.body.trim()));
  if (bare) return bare.body.trim();

  return null;
}

function neighborSpanFromIntent(intent: Chunk[], fallback: number): number {
  const text = intent.map((c) => `${c.title}\n${c.body}`).join("\n");
  const m = /(neighbors|span|window)\s+(\d+)/i.exec(text);
  if (!m) return fallback;
  const n = Number(m[2]);
  if (!Number.isFinite(n)) return fallback;
  return Math.max(0, Math.min(5, n));
}

export function run_JOURNAL_PLACEMENT_CONFIRM(args: {
  rc: RunContainer;
  ctx: JournalPlacementCtx;
}): RunContainer {
  const { rc, ctx } = args;

  const keystone_id = parseKeystoneIdFromTargets(rc.inputs.Targets);
  if (!keystone_id) {
    const outputs: RunOutputs = {
      chunks: [
        {
          id: `jp_${Date.now()}_${Math.random()}`,
          title: "Placement not found",
          body: "No keystone_id was provided in Targets, so placement could not be confirmed.",
          kind: "TEXT",
        },
      ],
      patch_segments: [],
      seals: [],
    };
    return { ...rc, outputs };
  }

  const all = ctx.chronicle.listAll();
  const idxFromEnd = [...all].reverse().findIndex((e) => e.keystone_id === keystone_id);

  const found = idxFromEnd >= 0;

  if (!found) {
    const resultKv =
      `run_id: ${rc.run_id}\n` +
      `keystone_id: ${keystone_id}\n` +
      `found: false\n` +
      `entry_id: ∅`;

    const outputs: RunOutputs = {
      chunks: [
        { id: `jp_${Date.now()}_${Math.random()}`, title: "ChroniclePlacementResult", body: resultKv, kind: "TEXT" },
        {
          id: `jp_${Date.now()}_${Math.random()}`,
          title: "Matched Entry",
          body: `No chronicle entry exists for keystone_id ${keystone_id}.`,
          kind: "TEXT",
        },
      ],
      patch_segments: [],
      seals: [],
    };
    return { ...rc, outputs };
  }

  const idx = all.length - 1 - idxFromEnd;
  const entry = all[idx];

  const span = neighborSpanFromIntent(rc.inputs.Intent, 1);
  const start = Math.max(0, idx - span);
  const end = Math.min(all.length, idx + span + 1);

  const matchedLine =
    `[${entry.school}] ${entry.journal_line}` +
    (entry.anchors.length ? ` — ${entry.anchors.join(", ")}` : "");

  const neighborhood = all
    .slice(start, end)
    .map(
      (e) =>
        `[${e.school}] ${e.journal_line}` +
        (e.anchors.length ? ` — ${e.anchors.join(", ")}` : "")
    )
    .join("\n");

  const resultKv =
    `run_id: ${rc.run_id}\n` +
    `keystone_id: ${keystone_id}\n` +
    `found: true\n` +
    `entry_id: ${entry.entry_id}`;

  const chunks: Chunk[] = [
    { id: `jp_${Date.now()}_${Math.random()}`, title: "ChroniclePlacementResult", body: resultKv, kind: "TEXT" },
    { id: `jp_${Date.now()}_${Math.random()}`, title: "Matched Entry", body: matchedLine, kind: "TEXT" },
    { id: `jp_${Date.now()}_${Math.random()}`, title: "Neighborhood", body: neighborhood, kind: "TEXT" },
    {
      id: `jp_${Date.now()}_${Math.random()}`,
      title: "Match Basis",
      body: `Matched by keystone_id reference: ${keystone_id}.`,
      kind: "TEXT",
    },
  ];

  const outputs: RunOutputs = { chunks, patch_segments: [], seals: [] };
  return { ...rc, outputs };
}
